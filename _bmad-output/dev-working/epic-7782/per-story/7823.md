## Downstream contracts locked (S2/S3/S4/S6/S8 must not break)

- **RPC signature:** `assign_next_pm(p_submission_id uuid) returns (assignment_kind text, team_member_id uuid, pm_first_name text, pm_photo_url text)`.
- **Idempotency key:** `submissions.pm_user_id is not null` (functionally equivalent to `referral_code` because `submissions.referral_code` is UNIQUE).
- **Error signal:** `E6_NO_ACTIVE_PMS` SQLSTATE `P0001` when no eligible PMs (active=true AND 'pm'=any(role)).
- **Eligibility rule:** a row in `team_members` is PM-eligible iff `active = true AND 'pm' = any(role)`.
- **TS types:** `TeamMemberRow`, `ProfileRow`, `SubmissionRow`, `SubmissionOfferRow`, `AssignmentEventRow`, `NotificationLogRow`, `AssignNextPmResult` — all exported from `src/lib/supabase/schema.ts`.
- **Env surface:** `SUPABASE_URL`, `SUPABASE_SERVICE_ROLE_KEY`, `SUPABASE_ANON_KEY`, `RESEND_API_KEY`, `EMAIL_FROM`, `EMAIL_REPLY_TO`, `EMAIL_CONTACT_WINDOW_HOURS`, `PM_ASSIGN_TIMEOUT_MS`.
