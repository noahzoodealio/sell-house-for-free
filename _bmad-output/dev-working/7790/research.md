## `LayoutProps<'/(marketing)'>` typegen

- Current `.next/types/routes.d.ts` (pre-S5) has `LayoutRoutes = "/"` — only root layout exists.
- Next 16 typegen regenerates `LayoutRoutes` + `ParamMap` + `LayoutSlotMap` on every `next dev` / `next build` / `next typegen`.
- Per story + architecture, typegen includes route-group folders in the route key: `'/(marketing)'`, `'/(legal)'`. Verified form: `LayoutProps<'/(marketing)'> = { params: Promise<ParamMap['/(marketing)']>, children: React.ReactNode } & { [K in LayoutSlotMap['/(marketing)']]: React.ReactNode }` — no dynamic params in these segments so `params` resolves to `{}`.
- `get-started/page.tsx` does NOT need `PageProps<>` — it has no params and no `searchParams` access; it can be a zero-prop Server Component.
